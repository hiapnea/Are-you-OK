**阿油 OK** 是一款专注于**独居长辈睡眠呼吸安全**的 iOS App。它通过智能算法识别睡眠中的呼吸暂停风险，并支持家人数据同步，让你能远程守护长辈的夜间安眠。

> 💡 本项目为旨在用技术缩短关爱距离。

## ✨ 核心功能

| 功能 | 说明 |
|-----|------|
| 🛑 呼吸暂停提醒 | 夜间自动识别潜在呼吸暂停事件，及时记录并推送 |
| 📊 智能睡眠记录 | 分析睡眠结构，输出总时长、深睡比例、醒来次数 |
| 📈 历史趋势图表 | 保存每次睡眠报告，形成健康变化趋势 |
| 👨‍👩‍👧 家人数据同步 | **独家特色**：长辈睡眠报告自动同步至家人手机 |
| 🔔 异常状态通知 | 发现夜间异常状况时，第一时间通知紧急联系人 |
| 👵 长辈模式 | 超大字体 + 极简信息，长辈也能轻松查看 |
